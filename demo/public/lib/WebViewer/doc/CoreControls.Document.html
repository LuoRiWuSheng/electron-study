Class: Document

CoreControls. Document

Represents a document with individual pages (canvases) that can be displayed on screen and printed.

new Document()

⚠ You must NOT instantiate this yourself. Access the document instance as follows:

Examples
// In your app
var viewer = new PDFTron.WebViewer({ ... }, viewerElement);
viewerElement.addEventListener('ready', function() {
  var instance = viewer.getInstance();
  var docViewer = instance.docViewer;
  var doc = docViewer.getDocument();
  // doc.someAPI();
});
// In a config file
$(document).on('viewerLoaded', function() {
  var instance = readerControl;
  var docViewer = instance.docViewer;
  var doc = docViewer.getDocument();
  // doc.someAPI();
});
