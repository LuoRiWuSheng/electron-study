Instructions

Use the dropdown above to view local or remote XOD documents. XOD is a web optimized XPS format, specifically designed for fast and efficient web vieweing.

You can convert a wide range of document formats (including PDF) into XOD using the PDFTron SDK. When converting, you can add a password to encrypt it for extra security. See XOD and encryption for details.
